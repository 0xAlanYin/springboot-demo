## 注意事项

### 指定 redis 版本启动

    /usr/local/Cellar/redis/5.0.3/bin/redis-server redis.conf

### 启动 sentinel 命令
   
   /usr/local/Cellar/redis/5.0.3/bin/redis-server redis.conf --sentinel
